# remote-db-copy

Remote Shell DB Copy

## How to create a copy

help: db-copy -h

Create a copy of the database for loading a new site

How to use: db-copy -d {TARGET_DBNAME} -f {BACKUP_DBPATH} -u {DB_USER}

End of help command
